# HR Analytics Dashboard – Excel

This project demonstrates an interactive HR Analytics dashboard built using Microsoft Excel.

# Features:

Employee distribution analysis

Attrition rate insights

Salary comparison by department

Monthly hiring trends

Performance rating breakdown

# Tools Used:

Excel Pivot Tables

Slicers

Data visualization techniques

# Objective:

To transform raw HR data into meaningful insights and build a structured, decision-focused dashboard.

Future Enhancement:

Rebuilding the same dashboard in Power BI to improve scalability, interactivity, and data modeling.
